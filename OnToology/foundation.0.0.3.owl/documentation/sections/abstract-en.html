Abstract

This Ontology describes the Foundation TV Series. It is based on the work "A conceptual model for an OWL ontology to represent the knowledge of transmedia storytelling". In this paper the authors have developed an OWL ontology from the conceptual model, whose namespace (also valid for downloading) is http://purl.org/umu/tso/ and its documentation is available at http://skos.um.es/TR/tso/.
